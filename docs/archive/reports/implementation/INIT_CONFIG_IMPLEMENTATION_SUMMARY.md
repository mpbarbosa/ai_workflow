## Features

### ✅ 5-Step Wizard Process

**Step 1**: Technology Stack Detection
- Auto-detects from project files
- Shows confidence score
- Manual selection fallback

**Step 2**: Project Information
- Project name (auto-detected)
- Optional description

**Step 3**: Project Structure
- Source directories
- Test directories
- Documentation directories

**Step 4**: Build & Test Commands
- Test command
- Lint command

**Step 5**: Review & Save
- Preview complete configuration
- Confirm before saving
- Generates `.workflow-config.yaml`

### ✅ User Experience

- **Colorful UI**: Cyan, green, yellow, red color scheme
- **Clear Progress**: "Step X/5" indicators
- **Smart Defaults**: Language-specific defaults
- **Safe**: Warns before overwriting existing config
- **Cancellable**: Can exit at any step

### ✅ Language Support

Supports all 8 languages with appropriate defaults:
- JavaScript/Node.js
- Python
- Go
- Java
- Ruby
- Rust
- C/C++
- Bash/Shell

## Usage

### Basic Usage
```bash
cd /path/to/project
./execute_tests_docs_workflow.sh --init-config
```

### With Target
```bash
./execute_tests_docs_workflow.sh --target /path/to/project --init-config
```

### Help
```bash
./execute_tests_docs_workflow.sh --help | grep init-config
# Output: --init-config       Run interactive configuration wizard
```
